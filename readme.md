# Dawg

## Usage
Just add the dependency to your `build.gradle`:

```groovy
dependencies {
    compile 'org.chalup:dawg:x.y.z'
}
```

Or `pom.xml`:

```xml
<dependency>
  <groupId>org.chalup</groupId>
  <artifactId>dawg</artifactId>
  <version>x.y.z</version>
</dependency>
```
